Descripción del proyecto

Japo es una marca ficticia, enfocada y orientada a la creacion de packagin, se tomó como referencia la cultira asiatica, aúnque está más orientada a Japón. Para este proyecto buscamos inspiración para crear tanto el logotipo como la paleta de colores y lo que serían las ilustraciones representativas, en este caso las nubes, aun así no nos limitamos solo a eso e hizimos algunas otras ilustraciónes para los mockups, tambien creamos un maquetado para la página web, un eslogan y la papeleria que seria utilizada por la marca. Para llevar a cabo las ideas utilizamos los programas de figma, para bocetar la pagina web, krita para hacer las ilustraciones y los mockups de los productos, canva para poder crear la presentación e ilustrator para crear el logotipo y el mockup de la papeleria.

Cliente

Japo

Tiempo de desarrollo

3 semanas
